# File-Based Contact Book (Python)

A menu-driven contact book application built using Python.

## Features
- Add, view, and search contacts
- Input validation for phone numbers
- Persistent storage using file handling
- Data retained after program restart

## Concepts Used
- Dictionaries
- File I/O
- Functions
- Exception handling
